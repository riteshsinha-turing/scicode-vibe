## Conventions

- One RFC per major design. Bug fixes and small changes don't need RFCs.
- Filename: `NNNN-short-slug.md` with zero-padded sequential ids.
- Status values: `Draft → Reviewing → Accepted → Implemented` (or `Rejected`, `Superseded`).
- Update the `Changelog` table at the bottom of each RFC for material changes.
- Reference upstream CTP docs and example canvases rather than duplicating content here.
